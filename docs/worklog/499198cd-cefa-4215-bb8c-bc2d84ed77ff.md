## 혼동행렬 재설계 내용

레이아웃을 **표준 confusion matrix 컨벤션**으로 변경:

```
                      AI
            ──────────────────────
            AI 통과        AI 실패
   ┌────────┬──────────┬──────────┐
사람 통과   │   ▒0▒    │     1    │
            ├──────────┼──────────┤
사람 실패   │    1     │   ▒0▒    │
            └──────────┴──────────┘
            ──────────────────────
                                  HUMAN

  ● AI 통과 = 사람 통과  0      ● 불일치  2
```

### 핵심 변화
- **축 라벨**: 양쪽 화살표(`← AI →`) 제거 → 깔끔한 `AI` / `HUMAN` 가이드 텍스트로 위·아래 분리
- **열 헤더**: 셀 위에 `AI 통과` / `AI 실패` 일반 텍스트 (small-caps 아님)
- **행 헤더**: 셀 좌측에 `사람 통과` / `사람 실패` 우측 정렬 텍스트
- **셀 내부**: 텍스트 라벨 제거, **숫자만** (Geist Mono `text-4xl tabular-nums`)
- **셀 비율**: `aspect-square`로 정사각형 → 진짜 matrix 모양
- **합의 셀(대각선, PP+FF)**: 은은한 `bg-muted/40` 채움
- **불일치 셀(PF+FP)**: `#ef4444` 좌측 2px 액센트만
- **빈 셀(value=0)**: `text-muted-foreground/40`으로 흐리게
- **하단 범례**: `● 합의 N    ● 불일치 N` 두 카테고리 명확히

세로 회전 텍스트(`writingMode`)도 제거해서 깔끔하게 정렬됐습니다. 다시 확인해보시고 어색한 곳 있으면 말씀해 주세요.

## 정리

### 어노테이션 필터 UI 변경

**전**: 우상단 작은 select dropdown (`<select><option>...</option></select>`)
- 첫 사용자가 발견하기 어려움
- 어떤 어노테이션에 몇 건 있는지 알 수 없음

**후**: 페이지 상단 라벨 + pill row
```
어노테이션 필터
[ 전체 · 3 ] [ hallucination · 2 ] [ qa_correctness · 0 ] [ guardrail · 1 ]
                                                              ^ 활성
```
- `text-[10px] uppercase tracking-[0.18em] text-muted-foreground` 라벨로 명시적으로 "이게 필터다" 알려줌
- 각 pill = "eval 이름 + 카운트"
- 활성: `border-foreground bg-foreground text-background` (역상)
- 비활성: `border-border bg-card hover:bg-muted`
- "전체" pill로 모든 eval 통합 보기

### 카운트 의미
각 pill의 숫자는 그 필터를 선택했을 때 **불일치 목록에 표시될 trace 수**:
- 페어가 있고 불일치도 있는 eval: 불일치 trace 수
- 페어 없고 사람 평가만 있는 eval: 사람 평가 trace 수 (fallback)
- `__all__`: 전체 통합 카운트

### 버그 수정
`humanReviewedTraceIds`가 `selectedEval`을 무시하던 문제 — 이제 선택된 eval로 필터링. 예: "guardrail" 선택 시 사람 평가가 guardrail에만 달린 trace만 fallback 리스트에 포함.

### 기본값
이전엔 useEffect로 첫 eval 자동 선택했는데, 이제 "전체"가 디폴트 (`selectedEval === ""`). 첫 인상이 더 자연스러움.
