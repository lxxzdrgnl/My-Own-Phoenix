## 🤖 Claude

LibreOffice Writer 옵션에서 이미지 기본 앵커를 바꿀 수 있어요.

**경로:** 도구(Tools) → 옵션(Options) → LibreOffice Writer → 서식 지원(Formatting Aids) → **이미지(Image)** 섹션의 **앵커(Anchor)** 드롭다운을 **문자로(As Character)** 변경 → 확인.

이후 삽입(Insert) → 이미지로 넣는 모든 그림이 기본적으로 "문자로" 앵커가 됩니다.

> 참고: 버전이 7.0 이전이거나 일부 배포판에서는 이 옵션이 없을 수 있어요. 그 경우엔 매크로(Tools → Macros)로 `Insert > Image` 동작 직후 `oShape.AnchorType = com.sun.star.text.TextContentAnchorType.AS_CHARACTER` 처리하거나, 한 번 삽입한 뒤 우클릭 → 앵커 → 문자로를 수동으로 바꿔야 합니다.
